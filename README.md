# 수호 메세지 서랍 (Smart SMS Guardian) 🛡️

우리 아이를 위한 안전하고 스마트한 기본 문자(SMS) 앱입니다. 
아이들이 유해한 스팸 문자, 피싱, 욕설 등으로부터 보호받으며 안전하게 소통할 수 있도록 돕습니다.

## 🌟 주요 기능

* **강력한 AI 스팸 필터링 (OpenAI 연동)** 🤖
  * 연락처에 없는 모르는 번호로 온 문자는 AI(GPT)가 즉시 분석합니다.
  * 도박, 성인물, 폭력/협박, 욕설, 대출/주식 등 유해한 내용이 포함된 경우 자동으로 스팸으로 분류하여 아이의 메인 화면에 노출되지 않게 차단합니다.
* **`[Web발신]` 즉시 차단** 🚫
  * 광고성 문자에 주로 포함되는 `[Web발신]` 키워드가 있는 문자는 AI 분석 없이 즉시 스팸함으로 이동하여 불필요한 알림을 방지합니다.
* **보호자 전용 스팸 보관함 (비밀번호 보호)** 🔒
  * 차단된 스팸 메시지는 부모님(보호자)만 확인할 수 있습니다.
  * 우측 상단 메뉴에서 '스팸 보관함' 진입 시 4자리 비밀번호(기본: 2026)를 입력해야 합니다.
* **텔레그램(Telegram) 스타일의 세련된 UI** 🎨
  * 메인 테마, 말풍선, 색상 등을 텔레그램 스타일의 쿨한 블루 컬러로 통일하여 친숙하고 세련된 대화 환경을 제공합니다.
* **편리한 메시지 전송 및 연락처 연동** ✉️
  * 연락처 앱과 연동되어 손쉽게 받을 사람을 검색하고 추가할 수 있습니다.
  * 장문 문자(LMS) 분할 전송 기능 및 사진 첨부(기기 기본 공유 시스템 연동)를 지원합니다.
* **쟈비스(Jarvis) AI 대화 기능** 🤖✨
  * 연락처에 "쟈비스"를 등록하거나, 수신자에 "쟈비스"를 입력하고 문자를 보내면 과금 없이 AI와 실시간으로 대화할 수 있습니다. 
  * 현재 날짜와 시간 인지 기능 탑재.

## 🚀 설치 및 설정 방법 (매우 중요)

### 1. OpenAI API 키 설정 (필수)
안드로이드 스튜디오에서 코드를 빌드하기 전, **반드시 아래 파일에 본인의 API 키를 입력**해야 AI 필터링과 쟈비스 기능이 정상 동작합니다.
1. `app/src/main/java/com/nolan/guardian/ContentFilter.kt` 파일을 엽니다.
2. 18번째 줄의 `API_KEY` 값을 본인의 실제 키로 변경합니다.
   ```kotlin
   private const val API_KEY = "sk-..." // 발급받은 실제 키 입력
   ```
*(※ GitHub 등 공개 저장소에 소스 코드를 올릴 때는 절대 실제 API 키를 그대로 올리지 않도록 주의하세요!)*

### 2. 기기에 앱 설치 및 기본 설정
1. 안드로이드 스튜디오의 상단 툴바에서 **Run 'app' (▶)** 버튼을 눌러 스마트폰에 빌드 및 설치를 진행합니다.
2. 기기에서 앱을 최초 실행하면 **'기본 SMS 앱'** 설정 안내가 나타납니다.
3. 팝업에서 **'수호 메세지 서랍'**을 기본 문자 앱으로 선택해야만 스팸 문자를 가로채고 정상적으로 수/발신할 수 있습니다.
